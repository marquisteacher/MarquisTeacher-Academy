## 📸 Overview

MarquisTeacher Academy is a full-stack EdTech platform built from the ground up by **Marquis Williams**. It combines a professional landing page, a 20-question English level assessment, and a community-driven exam leaderboard — all connected to a live Firebase backend.

---

## ✨ Features

### 🎯 English Level Assessment
- 20 carefully crafted questions across 4 skill areas
- Grammar · Vocabulary · Reading Comprehension · Idioms & Expressions
- Weighted scoring mapped to CEFR levels (A1 → C2)
- Randomised question order on every attempt
- Skill breakdown results with visual progress bars
- No pass or fail — just honest, clear results

### 🏆 Community Exam Board
- Live leaderboard powered by Firebase Firestore
- Persistent across all devices and users worldwide
- Self-assessment vs result comparison
- CEFR points scale: A1=10 · A2=20 · B1=40 · B2=60 · C1=80 · C2=100
- Gold 🥇 Silver 🥈 Bronze 🥉 medals for top performers

### 🤖 Marq the Mascot
- Animated guide that pops up throughout the experience
- Reacts to correct/incorrect answers in real time
- Celebrates results and encourages next steps
- Dismissable speech bubbles

### 📋 Registration Flow
- Name and self-assessment level captured before exam
- Results personalised with student's name
- Comparison between self-assessed and actual level

### 📬 Contact & Signup
- Direct email integration to MarquisTeacher@gmail.com
- Free consultation and signup options
- Pre-filled email subjects for smooth communication

## 🚀 Deployment

This site is deployed via **GitHub Pages** directly from the `main` branch.

To run locally:
1. Clone the repository
2. Open `index.html` in your browser
3. No build step required — pure HTML/CSS/JS
